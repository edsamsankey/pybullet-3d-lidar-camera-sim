# PyBullet 3D LiDAR + Camera Sensor Simulation

A robotics simulation project built using **PyBullet** that demonstrates a simple robot equipped with:

- 3D LiDAR (multi-ring raycasting)
- RGB Camera feed**
- OpenCV visualization**
- Debug visualization of LiDAR rays in the PyBullet GUI

This project mimics a basic sensor suite used in autonomous robotics.
